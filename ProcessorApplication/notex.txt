3.  **Run the Compiler (The Final Command):**
    Go back to your terminal (still in the project root) and run this command:

    ```bash
    npx tailwindcss -i ./wwwroot/css/input.css -o ./wwwroot/css/output.css --watch
    ```

    * `-i`: Specifies your **input** file.
    * `-o`: Specifies the **output** file. This is the *only* CSS file you'll need in your layout.
    * `--watch`: This is the best part. It keeps the command running and automatically re-compiles your `output.css` file *every time you save a change* to your `input.css` or any of your `.cshtml` files.

You should see it build successfully. A new file, `wwwroot/css/output.css`, will be created. **Leave this terminal window open while you are coding.**

## Phase 4: Update Your `_Layout.cshtml`

Now you just need to clean up your `_Layout.cshtml` to use your new, locally compiled file.

1.  Open `_Layout.cshtml`.
2.  **REMOVE** all of these:
    * The Tailwind CDN `<script src="https://cdn.tailwindcss.com"></script>`.
    * The Google Font `<link>` tags.
    * The `<script>` block with `tailwind.config = { ... }`.
    * The entire `<style>` block with `@font-face` (you moved it to `input.css`).
    * The entire `<style>` block with the loader/sidebar CSS (you also moved this to `input.css`).
3.  **ADD** this single line to link your *new* compiled CSS file:

    ```html
    <head>
        <meta charset="utf-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>@ViewData["Title"] - ProcessorApplication</title>
        
        <!-- Your other local CSS (if any) -->
        <link rel="stylesheet" href="~/ProcessorApplication.styles.css" asp-append-version="true" />
    
        <!-- 
          *** YOUR NEW COMPILED TAILWIND FILE ***
          This single file now contains Tailwind, your local "Inter" font,
          and all your custom styles.
        -->
        <link rel="stylesheet" href="~/css/output.css" asp-append-version="true" />
        
        <!-- Your local Font Awesome (this is correct) -->
        <link rel="stylesheet" href="~/lib/font-awesome/css/all.min.css" />
    </head>
